Multi-Utility Python App

 Overview

This project is a Multi-Utility Application built using Python. It combines multiple useful tools into a single program, making it efficient and user-friendly.

Features

The application includes:

- Calculator
- Unit Converter
- Temperature Convertor
- Physics Equation Solver

 Technologies Used

- Python 3
- Standard Python libraries (math etc.)

 How to Run

1. Install Python on your system
2. Download or clone this repository
3. Open terminal/command prompt
4. Run:
   python main.py

 How It Works

The program uses a menu-driven interface where the user selects different utilities. Each feature is implemented using functions and conditional statements.
